## WSL 설치

`PowerShell`에서 다음 명령어를 입력하여 WSL을 활성화한다:

```bash
wsl --install
```

## dfx 설치

`우분투 터미널`에서 다음 명령어를 입력하여 dfx를 설치한다:

```bash
sh -ci "$(curl -fsSL https://internetcomputer.org/install.sh)"
```

환경 변수를 적용한다:

```bash
source "$HOME/.local/share/dfx/env"
```

### dfx 버전 확인

다음 명령어를 입력하여 dfx 버전을 확인한다:

```bash
dfx --version
```
